# FlowFetch---YT-Playlist-Manager
A lightweight, modern Desktop Application built with Python and CustomTkinter to download entire YouTube playlists seamlessly with quality selection.
---

## ✨ Features
* **Modern UI:** Built using `CustomTkinter` with an elegant dark mode theme.
* **Asynchronous Downloading:** Utilizes Python's `threading` module to ensure the GUI remains responsive and never freezes during downloads.
* **Smart Playlist Organization:** Automatically creates a dedicated sub-folder for the playlist and numbers the videos sequentially (`01 - Title.mp4`).
* **Quality Control:** Allows users to choose their preferred maximum video resolution (1080p, 720p, 480p).
* **Robust Engine:** Powered by `yt-dlp` for high-speed and reliable media extraction.

---

## 🚀 Tech Stack
* **Language:** Python 3.x
* **GUI Framework:** CustomTkinter (Tkinter extension)
* **Core Engine:** yt-dlp
* **Concurrency:** Threading (Built-in)
